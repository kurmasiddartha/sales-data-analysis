# 📊 Sales Data Analysis

## 📌 Overview

This project performs data analysis on a structured sales dataset to identify trends, regional performance, and profit distribution. It demonstrates core data analysis skills including data cleaning, aggregation, and visualization.

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Google Colab

---

## 📂 Dataset Description

The dataset contains:

* Date of transaction
* Region (North, South, East, West)
* Product category
* Sales amount
* Profit

---

## 🔍 Key Features

* Data preprocessing and cleaning
* Region-wise sales and profit analysis
* Data aggregation using groupby operations
* Visualization of insights using bar charts and line graphs
* Monthly trend analysis

---

## 📈 Insights Generated

* Identified top-performing regions based on sales
* Compared profit distribution across regions
* Analyzed sales trends over time
* Observed relationships between sales and profit

---

## ▶️ How to Run

1. Open the notebook in Google Colab
2. Run all cells sequentially
3. View outputs and visualizations
